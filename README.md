# 로그인 기능 구현

## ✅ 구현 사항

- ID 입력 유효성 검사
  - 필수 입력
  - 이메일 형식 확인

- PW 입력 유효성 검사
  - 최소 8자 이상 입력

- blur 시 에러 메시지 노출
  - 입력 중에는 에러 메시지 숨김
  - 조건 충족 시 에러 메시지 제거

- 로그인 버튼 활성화 조건 적용
  - ID와 PW 모두 유효할 때만 활성화

- 에러 메시지 시각적 표시
  - 잘못된 입력 시 빨간 밑줄 표시

- 커스텀 훅 `useLoginForm` 분리
  - 상태 및 로직을 훅으로 관리
  - 컴포넌트에서는 useState 미사용

- 이메일 유효성 검사에 `validator.js` 사용
